Cull Diary — how the app will look after the emoji update

One rule: the column “In the app after update” is the whole UI change — SVG icon + text/numbers together, exactly as a user would see it. There is nothing to pick between: we are not showing “icon OR text”; we are showing the combined control. The left column is only the old emoji for comparison. This file does not change the live app until you edit diary.html / diary.js.

How to read each table

  • Now — what the diary shows today (emoji font).
  • In the app after update — the same control, with a stroke SVG instead of the emoji, plus the same labels/values you already use.
  • Where in the diary — which screen or element it belongs to.

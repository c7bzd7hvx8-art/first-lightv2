Deer School — design direction preview

These are static mockups only (no app wiring). Each block shows a considerable visual upgrade you could adopt: clearer hierarchy, a guided study path, richer readiness feedback, lettered quiz options, and a results hero that reads like an outcome screen.

In this file
  • A — Dashboard · study path strip · donut readiness · unified dark study cards + light tools row
  • B — Quiz · stepped progress · floating question card · A–D option rows · dual rail actions
  • C — Results · large typographic score · outcome badge · “revise next” as first-class card
Concept A

Dashboard — guided path + richer readiness

Adds a three-step strip (Study → Practise → Exam) so the app feels like a course, not a menu. Readiness becomes a visual ring + numeric headline instead of a thin bar alone. Study modes share one dark “primary lane” language; tools stay compact in a light row for parity.

Concept B

Quiz — stepped flow + lettered answers

Replace a single thin progress bar with micro-steps (one segment per question chunk). Options read like an exam paper: A–D badges, more breathing room, selected state with a left accent (not only colour).

Bold banner directions

These are not small tweaks — each option changes layout, type, palette, and metaphor. Compare to A (your current green card). Open banner-ui-options-preview.html in a browser. Pick one direction (or a hybrid) and we can implement it in the app.

A Baseline — today’s First Light banner (familiar).
B HUD — monospace, cyan “instrument”, segmented readout, sharp geometry.
C Editorial — cream paper, gold rule, serif clock; feels like a print field card.
D Dawn hero — full-width sky gradient, giant sun, time as the headline.
E Split (E1) — four time states with full chrome (date, location, moon, activity); E2–E4 are layout-only.
F Glass (F1) — four time states, full chrome inside panel; F2–F4 are material-only.
G Swiss — black / white / one green accent, brutal grid, minimal timeline.
